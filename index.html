Editing

  • Click a lane to place / remove a note
  • Click + drag vertically to draw a hold note
  • Drag a hold's end to scale it · middle to move
  • 1–9 place a note in that column at the playhead

Selecting

  • Shift + drag box-select (hold Ctrl to add)
  • Ctrl + click a note to add / remove it
  • Ctrl+A select all · M mirror · Delete remove
  • Ctrl+C / V copy / paste · Ctrl+D duplicate
  • Alt + arrows nudge selection (time / column)

Navigation

  • Space play / pause · Home / End jump to start / end
  • ↑ / ↓ or scroll move by one snap
  • Ctrl+Z / Ctrl+Y undo / redo · Ctrl+S save (.osz)
  • Drag the bottom timeline to scrub

Test play

  • Test launches immersive play from the playhead
  • Default keys auto-map per column (e.g. 4K = D F J K)
  • P pause · R retry · Esc back to the editor

Difficulties

  • Setup tab → Add / Duplicate / switch / delete
  • All diffs share the song; .osz exports them together
